# E-Commerce Sales Dashboard Portfolio

This repository contains a complete workflow for **e-commerce sales data analysis** and a **Power BI dashboard**.

## Project Overview

- **Data Cleaning & Preparation:** Python scripts handle duplicates, missing values, date parsing, sales categories, discounts, and aggregations.
- **CSV Outputs:** Cleaned datasets are exported for analysis:
  - `cleaned_sales_data.csv`
  - `customer_summary.csv`
  - `monthly_sales.csv`
  - `cust_promo.csv`
- **Visualizations:** Charts generated using Seaborn:
  - Sales by category
  - Sales by customer
  - Monthly sales trend
- **Power BI Dashboard:** Use the CSV files to create interactive dashboards in Power BI (`sales_ecommerce_dashboard.pbix`).

- **Requirenments**
  - pandas
  - matplotlib
  - seaborn
  - Power BI

- **Insights**
  - Sales trends by month
  - Top-performing customers and categories
  - Low-value customer identification for targeted promotions
